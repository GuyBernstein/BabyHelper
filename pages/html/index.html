Watch a comprehensive walkthrough of the site's architecture, tooling choices, and the design decisions that shaped this project.


The event-driven architecture ensures that insights are generated in real-time as data flows through the system, providing parents with immediate, actionable recommendations.

System Architecture Overview

The application leverages a modern microservices architecture with event-driven communication, ensuring scalability and real-time responsiveness across all components.

  • FastAPI Backend: High-performance Python framework orchestrating all system interactions
  • Apache Kafka: Event streaming platform enabling real-time data synchronization
  • PostgreSQL & Amazon S3: Dual storage strategy for structured data and media files
  • Anthropic Claude Integration: AI-powered insights analyzing baby patterns via Kafka event processing
